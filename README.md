# 進捗どうですか？

## 概要
かんたんな進捗管理アプリです。締切と短いコメントと共にタスクを登録し、自分のリストを作ることができます。<br>
リストに登録したタスクはワンクリックで「進捗だめです」「進捗あります」「タスク完了！」の3段階にわけて管理することができます。<br>
また、タスクは登録すると自動的に締切順に並べ替えられます。<br>
